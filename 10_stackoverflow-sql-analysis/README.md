# Анализ данных StackOverflow с помощью SQL

## Описание проекта

В рамках проекта проведён анализ базы данных StackOverflow — крупнейшей платформы вопросов и ответов для программистов.

Исследование выполнено с использованием PostgreSQL и включает решение аналитических задач различной сложности: от простых выборок и агрегаций до оконных функций, ранжирования и анализа пользовательской активности.

## Цель проекта

Изучить структуру базы данных StackOverflow и применить SQL для решения бизнес- и продуктовых задач:

- анализ активности пользователей;
- исследование публикаций и голосов;
- работа с рейтингами и значками;
- использование оконных функций;
- расчёт накопительных метрик;
- анализ временных рядов.

## Стек технологий

- SQL
- PostgreSQL

## Структура данных

В проекте использовались следующие таблицы:

- users — информация о пользователях платформы;
- posts — вопросы и ответы пользователей;
- post_types — типы публикаций;
- votes — голоса за публикации;
- vote_types — виды голосов;
- badges — значки пользователей.

## Выполненные задачи

### Базовые SQL-запросы

- фильтрация данных по условиям;
- работа с JOIN;
- агрегирование данных;
- использование подзапросов;
- группировка и сортировка результатов.

### Продвинутый SQL

- оконные функции (ROW_NUMBER, DENSE_RANK, LAG);
- накопительные метрики (SUM OVER);
- ранжирование пользователей;
- анализ активности по времени;
- расчёт пользовательских показателей.

### Аналитические исследования

- анализ просмотров публикаций по месяцам;
- поиск наиболее активных пользователей;
- исследование динамики публикаций;
- анализ пользовательской активности после регистрации;
- расчёт месячного изменения количества постов.

## Полученные навыки

В ходе проекта были отработаны:

- сложные JOIN-запросы;
- CTE (Common Table Expressions);
- оконные функции;
- анализ временных данных;
- построение аналитических SQL-запросов;
- работа с реальной реляционной базой данных.

## Результат

Проект позволил закрепить навыки написания SQL-запросов различной сложности и получить практический опыт решения аналитических задач на данных крупной пользовательской платформы.
