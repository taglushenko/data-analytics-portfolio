# Статистический анализ данных сервиса GoFast

## Описание проекта

Проект посвящён анализу данных сервиса аренды самокатов GoFast. В рамках исследования были изучены данные пользователей, история поездок и тарифные планы сервиса.

Основная цель проекта — исследовать поведение пользователей, рассчитать выручку и проверить гипотезы, которые могут помочь компании увеличить количество подписчиков тарифа Ultra.

## Цель исследования

* провести предобработку данных;
* изучить характеристики пользователей и поездок;
* сравнить пользователей с подпиской Ultra и без подписки;
* рассчитать ежемесячную выручку пользователей;
* проверить бизнес-гипотезы с использованием статистических методов;
* решить задачи, связанные с вероятностными распределениями.

## Стек

* Python
* Pandas
* NumPy
* Matplotlib
* SciPy

## Этапы работы

1. Загрузка и изучение данных.
2. Предобработка данных.
3. Исследовательский анализ данных.
4. Объединение таблиц.
5. Расчёт помесячной выручки.
6. Проверка статистических гипотез.
7. Анализ вероятностных распределений.

## Проверенные гипотезы

* Пользователи с подпиской Ultra проводят в поездках больше времени.
* Среднее расстояние одной поездки пользователей с подпиской не превышает оптимальное значение 3130 метров.
* Пользователи с подпиской приносят более высокую ежемесячную выручку.

## Основные результаты

* Большинство пользователей сервиса находятся в возрасте около 25 лет.
* Наиболее распространённая длина поездки составляет около 3 км.
* Пользователи с подпиской совершают более длительные поездки.
* Средняя длина поездки подписчиков соответствует оптимальному уровню эксплуатации самокатов.
* Подписчики Ultra обеспечивают более высокую выручку по сравнению с пользователями без подписки.
* Продвижение подписки Ultra может положительно влиять на финансовые показатели сервиса.
