# Анализ эффективности маркетинговых инвестиций Procrastinate Pro+

## Описание проекта

Проведён анализ эффективности маркетинговых инвестиций развлекательного приложения Procrastinate Pro+. Несмотря на значительные расходы на привлечение пользователей, компания столкнулась с проблемой низкой окупаемости рекламы.

Цель исследования — определить причины снижения ROI, оценить эффективность рекламных каналов и подготовить рекомендации по оптимизации маркетингового бюджета.

## Задачи исследования

* Проанализировать структуру пользовательской аудитории по странам, устройствам и каналам привлечения.
* Исследовать маркетинговые расходы и стоимость привлечения пользователей (CAC).
* Оценить показатели LTV, ROI, конверсии и удержания пользователей.
* Определить рекламные каналы, устройства и регионы, оказывающие наибольшее влияние на окупаемость рекламы.
* Подготовить рекомендации по повышению эффективности маркетинговых инвестиций.

## Используемые данные

В анализе использованы данные о пользователях приложения за период с 1 мая по 27 октября 2019 года:

* данные о посещениях пользователей;
* данные о покупках пользователей;
* данные о рекламных расходах.

## Используемые инструменты

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Когортный анализ
* Юнит-экономика
* Анализ маркетинговых метрик (LTV, CAC, ROI, Retention Rate, Conversion Rate)

## Основные результаты

* Реклама не окупается: итоговый ROI составляет около 81%.
* Основной причиной низкой окупаемости является высокая стоимость привлечения пользователей.
* Наибольшие маркетинговые расходы приходятся на каналы TipTop и FaceBoom, которые демонстрируют неудовлетворительную эффективность.
* Наиболее проблемным рынком являются США, где стоимость привлечения пользователей значительно выше, чем в других странах.
* Перспективными источниками привлечения оказались RocketSuperAds и lambdaMediaAds.

## Рекомендации

* Пересмотреть распределение рекламного бюджета между каналами.
* Сократить расходы на неэффективные источники привлечения.
* Сосредоточить инвестиции на каналах с устойчивой окупаемостью.
* Дополнительно исследовать причины низкого удержания пользователей по отдельным рекламным каналам.
