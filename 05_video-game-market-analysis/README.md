# Анализ рынка компьютерных игр

## Описание проекта

Исследовательский анализ рынка компьютерических игр на основе исторических данных интернет-магазина «Стримчик».

Цель проекта — выявить факторы, влияющие на успешность игр, определить перспективные платформы и жанры, а также подготовить рекомендации для планирования рекламной кампании на 2017 год.

## Используемые инструменты

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* SciPy
* Статистическая проверка гипотез

## Основные задачи

* Предобработка данных и работа с пропусками
* Анализ динамики продаж игровых платформ
* Выбор актуального периода для прогнозирования
* Исследование влияния оценок пользователей и критиков на продажи
* Анализ популярности игровых жанров
* Составление портрета пользователей разных регионов
* Проверка статистических гипотез

## Основные результаты

* Определён актуальный период для анализа рынка компьютерных игр.
* Выявлено, что жизненный цикл игровых платформ обычно составляет 7–10 лет.
* Наиболее перспективными платформами на конец рассматриваемого периода являются PS4 и Xbox One.
* Оценки критиков демонстрируют более заметную связь с продажами, чем пользовательские оценки.
* Наиболее прибыльными жанрами оказались Action и Shooter.
* Выявлены различия в предпочтениях пользователей Северной Америки, Европы и Японии.
* Статистически значимых различий между пользовательскими рейтингами Xbox One и PC не обнаружено.
* Подтверждено различие средних пользовательских рейтингов жанров Action и Sports.
