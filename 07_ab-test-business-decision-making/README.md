# Приоритизация гипотез и анализ результатов A/B-теста интернет-магазина

## Описание проекта

Проведён анализ гипотез по увеличению выручки интернет-магазина и исследованы результаты A/B-теста. Для выбора наиболее перспективных инициатив выполнена приоритизация гипотез с использованием фреймворков ICE и RICE. После этого проведён комплексный анализ результатов эксперимента с целью оценки влияния изменений на ключевые бизнес-показатели.

Цель исследования — определить наиболее приоритетные гипотезы для развития продукта и принять решение по итогам A/B-теста на основе статистического анализа данных.

## Задачи исследования

* Выполнить приоритизацию гипотез с помощью методов ICE и RICE.
* Сравнить результаты приоритизации и определить наиболее перспективные гипотезы.
* Проанализировать динамику кумулятивной выручки, среднего чека и конверсии по группам теста.
* Выявить аномальных пользователей и заказы.
* Оценить статистическую значимость различий между группами по «сырым» и «очищенным» данным.
* Принять решение о дальнейшей судьбе эксперимента на основе полученных результатов.

## Используемые данные

В анализе использованы данные интернет-магазина:

* данные о гипотезах по увеличению выручки с оценками Reach, Impact, Confidence и Efforts;
* данные о заказах пользователей в рамках A/B-теста;
* данные о количестве посетителей по группам теста.

## Основные результаты

* При использовании фреймворка RICE приоритет гипотез изменился по сравнению с ICE за счёт учёта охвата пользователей (Reach).
* В ходе подготовки данных были выявлены пользователи, попавшие одновременно в обе группы теста. Такие пользователи были исключены из анализа.
* Группа B на протяжении большей части теста демонстрировала более высокую конверсию по сравнению с группой A.
* Статистически значимые различия в количестве заказов на пользователя были обнаружены как по «сырым», так и по «очищенным» данным.
* Статистически значимых различий в среднем чеке между группами выявлено не было.
* Очистка данных от аномальных пользователей и заказов не изменила основные результаты исследования.

## Рекомендации

* Остановить A/B-тест и зафиксировать победу группы B.
* Рассмотреть внедрение изменений, протестированных на группе B.
* Продолжить мониторинг среднего чека после внедрения изменений, поскольку статистически значимых различий по данной метрике обнаружено не было.
* Использовать результаты приоритизации RICE при планировании дальнейших продуктовых экспериментов, поскольку метод учитывает потенциальный охват пользователей.
