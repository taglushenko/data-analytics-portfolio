# Исследование поведения пользователей Яндекс Музыки

## Описание проекта

В рамках проекта было проведено исследование данных сервиса Яндекс Музыка с целью сравнить музыкальные предпочтения пользователей Москвы и Санкт-Петербурга.

На основе данных о прослушиваниях были проверены гипотезы о различиях в активности пользователей и популярных музыкальных жанрах в двух городах.

## Цель исследования

Проверить следующие гипотезы:

1. Активность пользователей зависит от дня недели, причём в Москве и Санкт-Петербурге это проявляется по-разному.
2. Утром в понедельник и вечером в пятницу жители Москвы и Санкт-Петербурга предпочитают разные музыкальные жанры.
3. Москва и Санкт-Петербург отличаются по жанровым предпочтениям: в Москве чаще слушают поп-музыку, а в Санкт-Петербурге — русский рэп.

## Данные

Датасет содержит информацию о прослушиваниях пользователей Яндекс Музыки:

- user_id — идентификатор пользователя;
- track — название трека;
- artist — исполнитель;
- genre — музыкальный жанр;
- city — город пользователя;
- time — время начала прослушивания;
- day — день недели.

## Используемые инструменты

- Python
- Pandas
- Jupyter Notebook

## Этапы работы

### 1. Загрузка и изучение данных

Проведён первичный анализ структуры датасета, типов данных и наличия пропусков.

### 2. Предобработка данных

- приведены названия столбцов к единому стилю;
- обработаны пропущенные значения;
- удалены явные и неявные дубликаты.

### 3. Анализ данных

Для проверки гипотез были проанализированы:

- активность пользователей по дням недели;
- популярные жанры в разные дни и время суток;
- различия музыкальных предпочтений между городами.

### 4. Формулирование выводов

По результатам анализа были сделаны выводы относительно каждой из исследуемых гипотез.

## Результаты исследования

### Гипотеза 1

Активность пользователей зависит от дня недели и различается в Москве и Санкт-Петербурге.
Результат: подтверждена.

### Гипотеза 2

Музыкальные предпочтения пользователей различаются утром в понедельник и вечером в пятницу.
Результат: частично подтверждена.

### Гипотеза 3

В Москве чаще слушают поп-музыку, а в Санкт-Петербурге — русский рэп.
Результат: не подтверждена.
