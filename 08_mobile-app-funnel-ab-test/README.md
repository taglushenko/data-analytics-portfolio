# Анализ воронки продаж и A/A/B-теста мобильного приложения

## Описание проекта

Проведён анализ поведения пользователей мобильного приложения по продаже продуктов питания. В рамках исследования изучена воронка событий, определены этапы с наибольшим оттоком пользователей и проанализирован путь пользователя от первого взаимодействия с приложением до успешной покупки.

Также исследованы результаты A/A/B-теста, проведённого для оценки влияния изменения шрифта в интерфейсе приложения на пользовательское поведение. Для проверки корректности проведения эксперимента выполнен анализ контрольных групп, после чего проведено сравнение контрольных и экспериментальной групп.

Цель исследования — выявить основные точки потери пользователей в воронке продаж и определить, оказывает ли изменение шрифта статистически значимое влияние на ключевые пользовательские действия.

## Задачи исследования

* Подготовить данные для анализа и проверить их качество.
* Изучить пользовательскую активность и структуру событий в приложении.
* Определить период, пригодный для анализа, и исключить неполные данные.
* Построить воронку событий и рассчитать конверсию между её этапами.
* Определить этапы с наибольшим оттоком пользователей.
* Оценить долю пользователей, доходящих до успешной оплаты.
* Проверить корректность распределения пользователей между экспериментальными группами.
* Провести анализ результатов A/A-теста.
* Оценить статистическую значимость различий между контрольными и экспериментальной группами.
* Сформулировать выводы по результатам эксперимента и подготовить рекомендации.

## Используемые данные

В анализе использованы данные мобильного приложения:

* информация о действиях пользователей внутри приложения;
* уникальные идентификаторы пользователей;
* временные метки событий;
* данные о принадлежности пользователей к экспериментальным группам.

## Основные результаты

* После очистки данных для анализа был выбран период с полными и корректными логами событий.
* Построена воронка пользовательских событий от первого взаимодействия с приложением до успешной оплаты.
* Наибольший отток пользователей выявлен при переходе с главного экрана приложения к экрану предложений.
* До этапа успешной оплаты доходит менее половины пользователей, попадающих на первый этап воронки.
* Статистически значимых различий между контрольными группами A/A-теста обнаружено не было, что подтверждает корректность разбиения пользователей на группы.
* Изменение шрифта не оказало статистически значимого влияния на поведение пользователей и конверсию между этапами воронки.
* Результаты сохранились после учёта множественных статистических проверок.

## Рекомендации

* Изменение шрифта может быть внедрено без существенных рисков для пользовательских метрик, поскольку статистически значимого влияния на поведение пользователей не выявлено.
* Основное внимание рекомендуется уделить этапу перехода с главного экрана к экрану предложений, где наблюдается наибольшая потеря пользователей.
* Для поиска причин оттока пользователей на проблемном этапе целесообразно провести дополнительный продуктовый анализ и серию новых экспериментов.
* При проведении последующих экспериментов рекомендуется сохранять практику проверки корректности разбиения пользователей с помощью A/A-теста.
