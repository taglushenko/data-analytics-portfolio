# Исследование надёжности заёмщиков

## О проекте

Заказчик исследования — кредитный отдел банка.

Цель проекта — определить, влияет ли семейное положение, количество детей, уровень дохода и цель кредита на вероятность возникновения задолженности по кредиту.

Результаты анализа могут быть использованы при разработке модели кредитного скоринга для оценки кредитных рисков потенциальных заёмщиков.

## Ключевые результаты

✔ Заёмщики без детей реже допускают просрочки по кредитам.

✔ Клиенты, состоящие в официальном браке, демонстрируют более высокую платёжную дисциплину.

✔ Кредиты на операции с недвижимостью связаны с наиболее низкой долей задолженностей.

✔ Полученные закономерности могут использоваться как дополнительные признаки в моделях кредитного скоринга.

## Задачи исследования

В рамках проекта необходимо было:

- изучить структуру и качество исходных данных;
- обработать пропуски, дубликаты и аномальные значения;
- выполнить категоризацию признаков;
- исследовать факторы, влияющие на возврат кредита в срок;
- сформулировать выводы для бизнеса.

## Данные

Датасет содержит информацию о клиентах банка:

- количество детей;
- трудовой стаж;
- возраст;
- образование;
- семейное положение;
- тип занятости;
- ежемесячный доход;
- наличие задолженности по кредитам;
- цель получения кредита.

## Результаты исследования

### Количество детей

Клиенты без детей реже допускают просрочки по кредитам, чем клиенты с детьми.

### Семейное положение

Наиболее низкая доля задолженностей наблюдается среди клиентов, состоящих в официальном браке.

### Уровень дохода

Связь между доходом и вероятностью возникновения задолженности присутствует, однако различия между отдельными группами невелики.

### Цель кредита

Наиболее надёжными оказались клиенты, оформлявшие кредиты на операции с недвижимостью.

Наиболее высокий уровень задолженностей наблюдается среди клиентов, оформлявших кредиты на автомобиль и образование.

## Общий вывод

В ходе исследования была выявлена связь между характеристиками заёмщика и вероятностью возникновения задолженности по кредиту.

Наиболее заметное влияние на платёжную дисциплину показали:

- семейное положение;
- наличие детей;
- цель кредита.

Полученные результаты могут использоваться в качестве дополнительных факторов при построении модели кредитного скоринга и оценке кредитных рисков.
